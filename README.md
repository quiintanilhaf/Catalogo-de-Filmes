# 🎬 Catálogo de Filmes

Projeto simples desenvolvido com HTML e CSS.

## 🚀 Funcionalidades

- Página principal
- Imagens clicáveis
- Páginas individuais
- Descrição dos filmes
- Link para retornar
- Link de trailer
- Informações sobre criticas ao filme
- Navbar responsiva
- Estilização com CSS
  

---

## 💻 Tecnologias

- HTML5
- CSS3

---

## 📁 Estrutura

```bash
catalogo-filmes/
│
├── index.html
├── filme1.html
├── filme2.html
├── filme3.html
├── filme4.html
│
├── style.css
│
└── imagens/
```
